## 📖 About The Project

This assignment contains **11 Python programs** that demonstrate fundamental concepts including:

- **Loops**: `for` loops with `range()` function
- **Lists**: Creation, manipulation, and `len()` function
- **Tuples**: Mixed data types, tuple unpacking, nested tuples
- **String Iteration**: Character-by-character processing
- **Conditional Logic**: Even/odd number detection using modulo operator
- **User Input**: Collecting and processing user data

This is an excellent learning resource for beginners understanding Python's core data structures and loop constructs.
